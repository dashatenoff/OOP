# C++ OOP Practice

Учебный проект для практики объектно-ориентированного программирования на C++.

## Что реализовано
- Классы Vector, Point2D, Point3D
- Коллекция векторов
- Инкапсуляция, методы, работа с заголовочными файлами
- Сборка проекта с помощью CMake

## Технологии
C++, CMake, CLion

## Сборка
```bash
mkdir build
cd build
cmake ..
cmake --build .
